Numerical AM–FM Components

A Numerical AM–FM component AMFMcompN is a complex vector Vector{ComplexF64} which has be demodulated using the function AMFMdemod().

using ISA
using Plots

a₀(t) = exp(-t^2)
ω₀(t) = 25
φ₀ = 0
𝐶₀ = Tuple([a₀,ω₀,φ₀])
ψ₀ = AMFMcomp(𝐶₀)
a₁(t) = exp(-0.5t^2)
ω₁(t) = 10 + 3*sin(t)
φ₁ = 0
𝐶₁ = Tuple([a₁,ω₁,φ₁])
ψ₁ = AMFMcomp(𝐶₁)

t = -1.0:0.01:1.0

p1 = isaPlot3d(ψ₀,t)
p2 = isaPlot3d(AMFMmodel([ψ₀,ψ₁]),t)

#COMPONENT OBSERVATION
Ψ₀ = ψ₀(t)
Ψ₁ = ψ₁(t)

#ESTIMATE NUMERICAL COMPONENT
𝚿₀ = AMFMdemod(Ψ₀,t)
𝚿₁ = AMFMdemod(Ψ₁,t)

p3 = isaPlot3d(𝚿₀)
p4 = isaPlot3d([𝚿₀,𝚿₁])
plot(p1,p2,p3,p4)
